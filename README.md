# KonechnuiAvtomat

Конечные автоматы

Тип: Детерминированный конечный автомат (ДКА)

Пример: Варка зелья в игре



1) Тип автомата
 Детерминированный конечный автомат (ДКА) — каждый шаг варки однозначно определяет переход между состояниями.

2) Определение компонентов автомата
  1. Множество состояний Q:
     
     q0 — котёл пуст (начальное состояние);
     qbase - добавлена базовая основа;
     qherb1 - добавлен первый ингредиент: трава;
     qherb2 - добавлены две травы;
     qmineral - добавлен минерал;
     qstir - выполнено помешивание;
     qheat - выполнено нагревание;
     qready - зелье готово (конечное состояние);
     qerr - ошибка (нарушение рецепта).

  2. Входной алфавит Σ={B,H,M,S,T}, где:

     B — базовая основа (Base);

     H — ингредиент‑трава (Herb);

     M — минерал (Mineral);

     S — помешивание (Stir);

     T — нагревание (Temperature).

  3. Выходной алфавит не требуется (автомат распознающий).

  4. Функция переходов δ:Q×Σ→Q:

     δ(q0, B) - qbase;
     δ(qbase, H) - qherb1;
     δ(qherb1, H) - qherb2;
     δ(qherb2, M) - qmineral;
     δ(qmineral, S) - qstir;
     δ(qstir, T) - qheat;
     δ(qheat, S) - qready;
     все остальные переходы ведут в qerr.

  5. Начальное состояние q0.

  6. Конечные состояния F = {qready} — зелье сварено правильно.

  7. Состояние ошибки qerr — «поглощающее» (из него нет выходов).

5) Проверка работы автомата

  1. «B,H,H,M,S,T,S» (правильный рецепт):

  Начало в q0;
  
  «B» → qbase;
  
  «H» → qherb1;
  
  «H» → qherb2;
 
  «M» → qmineral;
 
  «S» → qstir;

  «T» → qheat;

  «S» → qready;

  Окончание в qready ∈ F;

  Результат: принята (зелье сварено!).



  2. «H,B,H,M» (неправильный порядок):

  Начало в q0;
  
  «H» → qerr(нельзя добавить траву до основы);

  Все последующие символы ведут в qerr;

  Окончание в qerr !∈ F;
  
  Результат: отклонена.



  3. «B,H,M» (пропущены шаги):

  Начало в q0;

  «B» → qbase;

  «H» → qherb1;

  «M» → qerr(минерал нельзя добавить после одной травы);

  Окончание в qerr !∈ F;

  Результат: отклонена.
  


  4. «B,H,H,M,S,S» (лишнее помешивание):

  Начало в q0;

  До «S» после минерала → qstir;

  Второй «S» → qerr(помешивать можно только после нагревания);

  Окончание в qerr !∈ F;
  
  Результат: отклонена.



  5. «B,H,H,M,T,S,S» (неправильный порядок нагревания):

  *
  
  Результат: отклонена.
